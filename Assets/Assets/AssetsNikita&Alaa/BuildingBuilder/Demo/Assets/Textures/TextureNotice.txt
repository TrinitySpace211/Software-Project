Texture Generation Notice

Please note that the building facade textures included in this demo folder were created using AI tools. They are provided strictly for demonstration purposes so you can see how the procedural structures look when fully populated.

The ground textures and tree textures included in this scene were NOT created using AI.

All of these demo textures are entirely optional, are not required for the core procedural generator to function, and can be safely deleted from your project at any time.
